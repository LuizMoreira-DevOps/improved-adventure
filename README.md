# 🚀 Somativa Dois

Aplicação desenvolvida em **React** com integração ao **Firebase**, contendo sistema de cadastro, autenticação de usuários e consulta de dados armazenados no Firestore.

## 🌐 Deploy Online

🔗 https://somativa-dois.web.app

---

## 📌 Objetivo do Projeto

Desenvolver uma aplicação React com múltiplas páginas utilizando **React Router DOM**, integrando autenticação e banco de dados em nuvem.

---

## 🧠 Tecnologias Utilizadas

### Front-end
- React
- Vite
- JavaScript (JSX)
- React Router DOM

### Backend as a Service
- Firebase Authentication
- Cloud Firestore
- Firebase Hosting

### Ferramentas
- Node.js
- npm
- Git
- GitHub

---

## 📄 Funcionalidades

### 👤 Página Cadastro

Permite cadastrar um novo usuário com os seguintes campos:

- E-mail
- Senha
- Nome
- Sobrenome
- Data de nascimento

### 🔐 Página Login

Permite autenticação via Firebase Authentication com:

- E-mail
- Senha

Se os dados estiverem corretos, o usuário é redirecionado para a página principal.

### 🏠 Página Principal

Exibe os dados do usuário autenticado:

- Nome
- Sobrenome
- Data de nascimento

---

## 🔀 Rotas da Aplicação

| Rota | Página |
|------|--------|
| / | Cadastro |
| /login | Login |
| /principal | Principal |
